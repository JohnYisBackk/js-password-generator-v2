# 🔐 Password Generator PRO

Modern password generator built with HTML, CSS and vanilla JavaScript.

Generate strong passwords with custom rules, live strength feedback and one-click copy support.

## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-password-generator-v2/

## ✨ Features

- 🔑 Generate random passwords
- 📏 Custom password length slider
- 🔡 Lowercase characters
- 🔠 Uppercase characters
- 🔢 Numbers
- ✨ Symbols
- 🚫 Exclude similar characters
- 📊 Password strength meter
- 🛡️ Security mode label
- 📋 Copy to clipboard
- ♻️ Reset to default settings
- 📱 Responsive modern UI
